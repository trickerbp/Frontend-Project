# EduMatch Resource Frontend

React + Vite frontend cho EduMatch Resource Mapping.

## Env

```bash
VITE_API_BASE_URL=http://localhost:8000
```

Production backend is configured in `.env.production`:

```bash
VITE_API_BASE_URL=https://backend-ntk-gudch7hxhzatard5.southeastasia-01.azurewebsites.net
```

## Run

```bash
npm install
npm run dev
```

Run frontend with `.env.production`:

```bash
npm run dev:prod
```

## Build

```bash
npm run build
npm run build:prod
```
